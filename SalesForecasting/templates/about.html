Project overview

Modern machine learning forecasting for revenue teams.

This application combines a production-ready Flask experience with a trained sales prediction model to provide practical, user-friendly forecasting for retail and operations teams.

The UI is intentionally crafted to feel like a polished SaaS product rather than a generic template, with responsive layouts, elegant motion, and thoughtful interaction details.

Technologies used

  • Flask
  • Python
  • Bootstrap 5
  • AOS animations
  • scikit-learn

Dataset

The forecasting demo uses a sales dataset with date, store, sales, promotion, and holiday signals.

Algorithm

A Random Forest regressor is used to estimate future sales based on the provided features.

Author

Built as a production-ready web app experience for showcasing ML in a practical business context.
